Entry 2 - Benny's List

This one wasn't too bad, but there were a couple of things that gave me trouble. Getting the results to the right of the filter box took some experimenting with absolute positioning, since I didn't find an easy way to position it relative to the filter box. The filter box itself was the hardest part for me. Getting the input fields to be on the same line as the labels wasn't bad, that was just setting them both to be inline elements. Trying to shrink the input fields to fit the width of the box stumped me for about an hour, since I didn't want to just maually specify a specific pixel width. I ended up using a flex-box to be able to have each input field be able to grow to fit the available width. the MDN had a couple of examples for how to set up a flexbox that were useful.
